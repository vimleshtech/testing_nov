There are two data types available in Java:

-Primitive Data Types
	Without Decimal 
	-byte	: 1 byte
	-short	: 2 byte
	-int	: 4 byte
	-long	: 8 byte

	With Decimal 
	-float	: 4 byte
	-double	: 8 byte
	
	Alpha Numeric:
	-char	: 2 byte 	char c ='1';	

-Reference/Object Data Types

	-ArrayList
	-List
	-HashMap
